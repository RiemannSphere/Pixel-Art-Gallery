How to use is:

  1. Open program and click "Enter gallery".
  2. Sidebar allows you to navigate between pages.
  3. Sidebar offers following options:
    1. LOAD -> loading external artwork
    2. MENU -> going back to welcome page
    3. DISPLAY -> presenting artworks in a table
    4. EXIT -> closing program
  4. On DISPLAY: comment and rate artworks by choosing artwork and clicking "Comment" button. Edit artwork using "Edit" button. Display artwork's image using "View" button. 
  5. On LOAD: click "Load File" to load your image, then provide info in given fields.
